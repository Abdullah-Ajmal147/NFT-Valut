Transaction Check API

API Endpoint

Use this endpoint to check the status of a transaction programmatically.

Endpoint
GET /api/transactions/check/
Parameters
  • transaction_hash (required): The hash of the transaction to check
Example Request
curl -X GET "https://your-domain.com/api/transactions/check/?transaction_hash=0x123..."
Example Response
{
    "status": "completed",
    "transaction_id": 123,
    "transaction_type": "buy",
    "nft_name": "Example NFT",
    "price": "1.5",
    "created_at": "2024-03-20T10:30:00Z"
}
Response Fields
  • status: The current status of the transaction (pending, completed, failed)
  • transaction_id: The unique identifier of the transaction
  • transaction_type: The type of transaction (buy, sell)
  • nft_name: The name of the NFT involved in the transaction
  • price: The price of the transaction in ETH
  • created_at: The timestamp when the transaction was created
Error Response
{
    "error": "Transaction not found"
}
